state - 
It is internal data of component. It is used to store data that changes over time.

props -
IT is external data of component .It is passed from parent compoenent to child component.

In React, props are short for "properties". They are the data that is passed from a parent component to a child component. Props are read-only, which means that the child component cannot change them. Props are used to pass data between components in React
